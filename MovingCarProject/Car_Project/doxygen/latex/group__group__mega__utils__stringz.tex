Stringize. 

Stringize a preprocessing token, this token being allowed to be \#defined.

May be used only within macros with the token passed as an argument if the token is \#defined.

For example, writing \mbox{\hyperlink{group__group__mega__utils__stringz_gab3a4df2ccbd03c2809ee7ad542ab9793}{STRINGZ(\+PIN)}} within a macro \#defined by PIN\+\_\+\+NAME(\+PIN) and invoked as PIN\+\_\+\+NAME(\+PIN0) with PIN0 \#defined as A0 is equivalent to writing \char`\"{}\+A0\char`\"{}.
